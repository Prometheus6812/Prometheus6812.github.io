# Heindrich Burger — Virtual CV

My personal portfolio website, built as a virtual CV and hosted on **GitHub Pages**.

🔗 **Live site:** https://YOURUSERNAME.github.io

## About

This site is my online CV. It introduces who I am, my experience, skills, and projects, and lets visitors download my full CV as a PDF or reach me directly.

## Sections

- **Home** — animated terminal-style introduction
- **About** — who I am and what I focus on
- **Experience** — interactive timeline of my work and leadership roles
- **Skills** — grouped skill cards (systems, business, people, ways of working)
- **Projects** — links to my GitHub repositories
- **Contact** — email, GitHub, and LinkedIn links

## Features

- ⌨️ Typed "terminal" intro animation on the home screen
- 🌗 Dark / light mode toggle (remembers your choice)
- 🎞️ Scroll-triggered reveal animations (IntersectionObserver)
- 🧭 Sticky navigation that highlights the section you're reading
- 📱 Fully responsive — works on phones, tablets, and desktops
- ♿ Respects `prefers-reduced-motion` for accessibility

## Built with

- **HTML5** — semantic structure
- **CSS3** — custom properties (variables), grid, flexbox, animations, `color-mix()`
- **Vanilla JavaScript** — no frameworks
- **GitHub Pages** — free hosting straight from this repository

## Running it locally

1. Clone or download this repository
2. Open `index.html` in any modern browser — no build steps or installs needed

## File structure

```
├── index.html   # page structure and content
├── style.css    # design, layout, themes, animations
├── script.js    # interactivity (typing, dark mode, reveals, menu)
├── cv.pdf       # downloadable CV
└── README.md    # this file
```
